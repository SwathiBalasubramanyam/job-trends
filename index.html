Job Trends
JobTrends is a data visualization site that displays current job trends based on a given title. , Users can enter title / location/ company in a searchable input box, upon which the website suggests top most skills listed for a given title or gives the top most employers for the given location or suggests companies hiring at a given location. The users will also be able to see any job postings that are listed for a given title/location/company.
